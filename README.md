# Бот опыта и ежедневнок

Бот, который позволяет получать опыт при общении, собрать ежеденевную награду и купить роль
!daily -- раз в день (дает (по умолчанию 100) монеты)
!buy - за (по умолчанию 50) N монет забирает указанную сумму и дает роль / пишет, если уже есть или нету денег.
При общении раз в 30 секунд дает за сообщение от 5 до 15 опыта
